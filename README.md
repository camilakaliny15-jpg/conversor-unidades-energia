# ⚡ Conversor de Unidades de Energia

Projeto de software web desenvolvido colaborativamente para fins acadêmicos. O sistema permite realizar a conversão de diversas grandezas de energia em tempo real, rodando puramente no lado do cliente (frontend).

## 🚀 Funcionalidades e Escopo do Tema
O sistema realiza a conversão dinâmica entre as seguintes unidades:
*   Joule (J)
*   Quilojoule (kJ)
*   Caloria (cal)
*   Quilocaloria (kcal)
*   Watt-hora (Wh)
*   Quilowatt-hora (kWh)

## 📌 Diferenças entre as Versões (Releases)
Conforme os critérios de avaliação exigidos:
*   **v1.0.0 (Lançamento Funcional):** Versão focada puramente no funcionamento e exatidão matemática dos cálculos de conversão, sem aplicação de estilizações complexas.
*   **v2.0.0 (Estética Elaborada):** Refatoração visual completa da interface utilizando CSS moderno, gradientes, sombras, melhoria na tipografia e layout responsivo focado na experiência do usuário.
